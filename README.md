This project mashups two unrelated data sources:  
1. **Weather data** (Open-Meteo API)  
2. **Netflix Top 10 daily popularity**

The dashboard shows interesting correlations between weather conditions and the genres people tend to watch.

## Features
- Live weather data for any city
- Netflix top-10 trends
- Correlation graphs
- Genre heatmaps
- Downloadable reports
